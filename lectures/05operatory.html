Operatory
  • budują wyrażenia ze zmiennych, literałów, wywołań funkcji
  • są ewaluowane (obliczane) a ich wartość jest wykorzystywana w innych wyrażeniach:
    • w instrukcjach sterujących (operatory logiczne, relacyjne, inkrementacji)
    • w obliczeniach matematycznych (operatory arytmetyczne)
    • przypisywana zmiennym i stałym (grupa operatorów przypisania)
  • argumenty operatora nazywamy czasami operandami
  • argumenty często są innymi wyrażeniami
Arność, priorytet, łączność
  • ilość argumentów to arność operatora, np.
    i++; operator inkrementacji, unarny
    x % 2; operator dzielenia modulo, binarny
    x > y ? 1 : 0; operator warunkowy, jedyny ternarny
  • operatory ewaluowane są w kolejności określanej przez priorytet
    x + y * 2; // x + ( y * 2 ) gdyż mnożenie ma wyższy priorytet niż dodawanie
  • gdy wyrażenie zawiera więcej operatorów o tym samym priorytecie, to kolejność ich obliczania określana jest przez łączność operatorów (prawo lub lewostronna)
    x / y * 2; // ( x / y ) * 2 gdyż operatory / i * są lewostronnie łączne tzn. wykonywane są od lewej do prawej

Prawda i fałsz w C
  • w języku C nie ma osobnego typu do reprezentowania prawdy i fałszu
  • w językach Java, C++, Pascal mamy typ boolean i literały true oraz false
  • niezerowe wartości wyrażeń reprezentują prawdę w C
  • zerowe wartości wyrażeń reprezentują fałsz w C
Operatory relacyjne
Operatory relacyjne porównują wartości swoich operandów, najczęściej występują w instrukcjach sterujących
x == 0
równy
x != 0
różny
x > y % 2
y * y >= 4
większy, większy lub równy
Bardzo częsty błąd, którego nie wykryje kompilator (instrukcja przypisania też zwraca wartość):
if ( x = 2 ) zamiast if ( x == 2 )

Operatory bitowe: przykład 
#include <stdio.h>

int main() {
    short int n = 0x00a4;   /* 00000000 10100100 */
    short int m = 0x00b7;   /* 00000000 10110111 */

    printf("n & m == %04x\n", n & m);
    printf("n | m == %04x\n", n | m);
    printf("n ^ m == %04x\n", n ^ m);
    printf("~n == %04x\n", ~n);
    printf("n << 3 == %04x\n", n << 3);
    printf("n >> 3 == %04x\n", n >> 3);
    return 0;
}
Operatory grupy przypisania
+= -= *= /= %= <<= ^= ...
przypisanie z innym operatorem binarnym
x += 3
x = x + 3
x op= wyr
x = x op wyr
Operator przypisania też zwraca wartość. Jest to wartość wyrażenia "po prawej" jego stronie.
Reguły łączności
  • operatory jednoargumentowe i grupy przypisania są prawostronnie łączne (od prawej do lewej)
    x = y = z; // x = ( y = z )
  • reszta operatorów jest lewostronnie łączna (od lewej do prawej)
    x + y + z; // ( x + y ) + z
